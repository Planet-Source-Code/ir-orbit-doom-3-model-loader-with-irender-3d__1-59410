Simple Doom 3 Model Loader with iRender 3D game Engine.

To run this application rename the following files:

	iRender.rename		to	iRender.dll
	ActorMD5Plug.rename	to 	ActorMD5Plug.dll
	Register.rename		to	Register.exe

Register.exe is the program that registers iRender 3D game engine dinamic library. ActorMD5Plug.dll if plugin to the engine that allows to render Doom III models.
